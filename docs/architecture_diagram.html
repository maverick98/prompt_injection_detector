End-to-End Architecture

The project is organized as a living defense pipeline. Prompts and retrieved content are scored by ML and mathematical risk signals, tested by adaptive attackers, then summarized in reports, notebooks, and the Streamlit application.

Information Theory

Scores leakage intent by combining secret-seeking language, hidden-state references, entropy, and compression pressure.

Leakage = f(secret terms, hidden state, H(text))

Statistical Physics

Turns override, exfiltration, obfuscation, tool pressure, and authority hijack into a single risk-energy landscape.

E = w dot signal_vector

Phase Transitions

Maps prompts into low-risk, critical review, and high-risk phases so thresholds behave like operating regions.

low -> critical -> block

Conformal Uncertainty

Returns an ambiguous prediction set when classifier score or energy is near a decision boundary.

Y_hat = {clean, injection}

Sequential Detection

Uses CUSUM-style accumulation to expose multi-turn attacks that look harmless one turn at a time.

S_t = max(0, alpha S_t-1 + E_t - baseline)

Graph, Transport, Control

Finds dangerous paths, compares distance to clean and attack prototypes, then recommends a control action.

path risk -> OT affinity -> guardrail
